1. Helm release for go-mail-webhook-service installed.

2. Configuration
   The application reads its configuration from:
     /go/config/config.yaml
   This file is rendered from the chart values:
     .Values.config (object)
   Edit your values and upgrade:
     helm upgrade {{ .Release.Name }} {{ .Chart.Name }} -f your-values.yaml

3. Execution Modes
   - Job:
     If .Values.job.enabled = true, a Helm hook Job will run after install/upgrade.
   - CronJob:
     If .Values.cronjob.enabled = true, a CronJob will run on the configured schedule.

4. Image configuration
   You can override the image used by setting:
     --set image.repository=your.registry/go-mail-webhook-service
     --set image.tag=yourTag

5. Uninstall
   helm uninstall {{ .Release.Name }}
