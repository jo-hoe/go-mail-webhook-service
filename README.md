Webhook allowing to pull mails and send requests to an callback url.

## Configuration Example

```yaml
- mail-service-config: 
    mail: example@gmail.com
    
  subject-regex: ".*"
  body-regex: ".*"
  callback-url: https://example.com/callback
  callback-method: POST
```
